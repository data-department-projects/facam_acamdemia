# Images de la page d'accueil

Fichiers utilisés par la page d'accueil (ne pas renommer si vous voulez garder l’affichage) :

- **`image hero.jpg`** — image de fond du hero (derrière le bloc bleu).
- **`maintenance.jpg`** — carte « Maintenance industrielle ».
- **`production.jpg`** — carte « Production & Logistique ».
- **`qhse.jpg`** — carte « QHSE - Qualité Sécurité ».

Format recommandé : JPG ou PNG, ratio 16/11 ou similaire pour les cartes programmes.
